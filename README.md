# Senshu-risyuu-Plus

専修大学の履修申請（Uniprove）の **履修画面** と **シラバス検索** を補助し、**UI/UX（モダン表示・可読性）** を改善する、非公式の Google Chrome 用拡張機能です。

**本拡張は大学・システム運営者とは一切関係ありません。** 利用は自己責任でお願いします。利用規約や試験時の注意事項に違反しないようご確認ください。

---

## できること（概要）

| 機能 | 説明 |
| --- | --- |
| **無操作キープ** | 設定した時間操作がないと、指定した操作を自動で行いセッション切れを避けやすくします（`ris.acc.senshu-u.ac.jp`）。 |
| **シラバス検索** | 配当科目選択画面から、条件を渡してシラバス検索を別タブで開きます（`syllabus.acc.senshu-u.ac.jp`）。ポップアップで曜日・時限や「1件目へ遷移」などを選べます。 |
| **モダンUI** | ポップアップと履修画面の見た目を、読みやすい配色・罫線・Noto Sans JP ベースに整えるモード（オフで従来表示に近い見た目）。 |

詳細な設定は、拡張のアイコンをクリックして開く **ポップアップ** から行います。

## GitHub に上げる（公開・バックアップ）

1. [GitHub](https://github.com) で新しい **Repository** を作成（空の README は付けなくてよい）。
2. このフォルダで次を実行します（初回のみ `git init`）。

```bash
cd Senshu-risyuu-Plus
git init
git add .
git commit -m "Initial commit: Senshu-risyuu-Plus Chrome extension"
git branch -M main
git remote add origin https://github.com/<あなたのユーザー名>/<リポジトリ名>.git
git push -u origin main
```

SSH を使う場合は `git remote add origin git@github.com:<ユーザー>/<リポジトリ>.git` に読み替えてください。

**クローンした人向け:** リポジトリを clone したあと、Chrome の「パッケージ化されていない拡張機能を読み込む」で **`chrome-extension` フォルダ** を指定すれば同じように使えます。

## モダンUIについて

- ポップアップで **「モダンな見た目」** のオン／オフを保存すると、次の表示がまとめて切り替わります。
  - 拡張の **ポップアップ**
  - **履修サイト**（一覧の表・ボタン・ヘッダ周りなど。`ris-page-modern.css` で上書き）
  - **デバッグ表示**（デバッグモードON時、画面右下のオーバーレイ）
  - 配当一覧に追加する **「シラバス」列と「検索」ボタン**
- 本文は濃いグレー、表は **セル罫線**、ボタンは **枠線** を付けて背景と区別しやすくしています。
- 日本語表示は **Noto Sans JP**（Google Fonts）を読み込み、フォント指定の先頭に置いています。履修ページ側で Content-Security-Policy によりフォントがブロックされる場合は、OS のフォールバックフォントになります。

## 拡張フォルダ内の主なファイル

| ファイル | 役割 |
| --- | --- |
| `manifest.json` | MV3 の定義（権限・コンテンツスクリプト・Web アクセス可能リソースなど） |
| `popup.html` / `popup.js` / `popup.css` | 設定UI |
| `content.js` | 無操作キープ、デバッグオーバーレイ、モダンUI用スタイルの注入 |
| `syllabus-search.js` | 配当画面への「シラバス」列・検索、学科名の保存 |
| `syllabus-fill.js` / `syllabus-results.js` | シラバス側のフォーム入力・1件目詳細への遷移 |
| `ris-page-modern.css` | 履修画面のモダン用スタイル（`content.js` から読み込み） |

## インストール方法（Chrome Web ストアを使わない場合）

Chrome ストアでは配布していないため、**フォルダをそのまま読み込む（パッケージ化されていない拡張機能）** という方法で入れます。初めての方は、次の順に進めてください。

### 0. 用意するもの

- **Google Chrome**（Windows / Mac どちらでも可）
  - **Microsoft Edge** でもほぼ同じ手順で利用できます（「拡張機能」画面の URL が `edge://extensions` になります）。
- このリポジトリの **`chrome-extension` フォルダ一式**[ここからダウンロードできます](https://github.com/yuumasyo/Senshu-risyuu-Plus/releases/tag/%E3%83%AA%E3%83%AA%E3%83%BC%E3%82%B9)。

> **重要:** インストール時に選ぶのは、リポジトリ全体ではなく **`chrome-extension` という名前のフォルダ** です(Releseからダウンロードした場合はそのまま使えます)。

### 1. 拡張機能の管理画面を開く

1. Chrome を開きます。
2. アドレスバーに次を入力して Enter します。  
   `chrome://extensions`
3. 「拡張機能を管理」という画面が開きます。

### 2. デベロッパーモードをオンにする

画面の **右上あたり** に **「デベロッパーモード」** というスイッチがあります。

1. それを **オン（右）** にします。
   - 初めての方へ: 「開発者向け」に見える名前ですが、**自分で入れたい拡張を読み込むために必要**な標準の手順です。

### 3. フォルダを読み込む

1. デベロッパーモードをオンにすると、画面上部に **「パッケージ化されていない拡張機能を読み込む」** というボタンが出ます。
2. そのボタンをクリックします。
3. フォルダ選択のダイアログが開くので、**`chrome-extension` フォルダ** を選びます（フォルダの中身ではなく、**`chrome-extension` そのもの** を指定）。
4. 「フォルダーの選択」などで確定します。

### 4. 入ったか確認する

- 拡張一覧に **「Senshu-risyuu-Plus（専修大学 履修画面・UI/UX）」** が表示され、スイッチがオンになっていれば成功です。
- 失敗すると「マニフェスト ファイルが読み取れません」などと出ることがあります。その場合は、**`manifest.json` が直接入っているフォルダ** を選び直してください（ひとつ上の親フォルダを選んでいないか確認）。

### 5.（任意）ツールバーにピン留めする

1. Chrome 右上の **パズルピースのアイコン**（拡張機能）をクリックします。
2. 一覧の **Senshu-risyuu-Plus** の右側にある **ピンのアイコン** を押すと、いつでもアイコンから設定を開けます。

## アップデートしたとき

GitHub から新しいファイルを取得したあと、**もう一度 `chrome://extensions` を開き**、Senshu-risyuu-Plus のカードにある **「更新」ボタン（丸矢印）** を押すか、一度 **オフ → オン** にすると読み込み直せます。うまく反映されないときは、拡張を **削除** してから、上記「3. フォルダを読み込む」をやり直してください。

**拡張を再読み込みした直後**は、すでに開いていた履修・シラバスのタブでは古いコンテキストのまま動くことがあります。**そのタブを更新（F5）** すると新しいスクリプトが載ります。

---

## 権限について

拡張が求める権限は **`storage`（データの保存）** のみです。設定内容を Chrome に保存するために使います。

`manifest.json` では、履修ページ向けに **`ris-page-modern.css`** を Web アクセス可能リソースとして登録しています（モダンUIでスタイルを読み込むため）。

実際には、次のサイトでのみスクリプトが動きます（`manifest.json` の指定どおり）。

- `https://ris.acc.senshu-u.ac.jp/*`（履修申請）
- `https://syllabus.acc.senshu-u.ac.jp/*`（シラバス）

モダンUIのフォントは **Google Fonts** へ HTTPS で取得します（ネットワーク利用）。

## トラブルシューティング

| 症状 | 試すこと |
| --- | --- |
| 拡張が一覧に出ない | `chrome-extension` フォルダを指定しているか、`manifest.json` がその直下にあるか確認。 |
| ボタンや動きが出ない | `chrome://extensions` で Senshu-risyuu-Plus を **再読み込み** し、**履修・シラバスのページを再表示（更新）** する。 |
| コンソールに `Extension context invalidated` | 拡張を再読み込みしたあと **ページを更新** していないことが多い。該当タブを **F5** する。 |
| シラバスでエラーになる | ポップアップの **「曜日・時限を含める」** をオフにして保存し直す。 |
| モダンUIのフォントが変わらない | 大学サイトの CSP で Google Fonts がブロックされている可能性。ポップアップは通常 Noto が当たる。 |
